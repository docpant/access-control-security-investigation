## Overview

This project analyzes a suspicious financial transaction within a business environment to identify weaknesses in employee access controls and permission management.

The investigation focuses on reviewing event logs, identifying access control vulnerabilities, and recommending mitigations to reduce the risk of future insider threats and unauthorized financial activity.

## Objective

- Review event logs
- Investigate suspicious account activity
- Identify access control weaknesses
- Analyze employee permissions
- Recommend security mitigations

## Skills Demonstrated

- Event log analysis
- Access control auditing
- Insider threat investigation
- Identity and access management (IAM)
- Security documentation
- Risk mitigation planning

## Project Structure

- project-description.md
- event-log-analysis.md
- access-control-issues.md
- security-recommendations.md
- summary.md

## Outcome

The project identified weaknesses in employee access management practices and proposed security improvements to strengthen organizational access controls and reduce future security risks.
